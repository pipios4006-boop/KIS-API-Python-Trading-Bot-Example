✨ 주요 기술적 특징 (Key Features)
💎 초경량 2대 코어 아키텍처 : 시스템 복잡도를 유발하던 구형 V_VWAP 및 V17/V13 레거시 모드를 100% 영구 소각하고, 순수 공격형 V14(오리지널 무매)와 순수 방어형 V-REV(역추세) 단 2개의 핵심 코어만으로 작동하도록 아키텍처를 전면 다이어트했습니다.
💎 V-REV 층별 고유 단가 독립 탈출 (LIFO) : 기존의 전체 평단가 의존 로직을 폐기하고, 각 매수 지층(Layer)이 본연의 고유 진입 단가를 기억하여 조건 충족 시 독립적으로 개별 탈출하는 후입선출(LIFO) 아키텍처를 완벽 이식했습니다.
💎 수동 긴급 수혈 (Emergency MOC) 시스템 : 통신 지연에 따른 자동 스케줄러의 오판 및 강제 청산 엣지 케이스를 원천 차단하기 위해 자동 청산 스케줄러를 영구 적출했습니다. 포트폴리오 매니저의 개입 하에 텔레그램 큐(Queue) 관리 메뉴에서 3중 경고 확인을 거쳐 최근 물량(LIFO)을 시장가(MOC)로 강제 매도하여 안전하게 실탄을 확보합니다. 장운영시간(프리장/정규장) 외 격발은 원천 차단됩니다.
💎 잭팟 및 1층 전량 강제 청산 (Sweep Finisher) : 장 마감 2분 전(15:58 EST), 잭팟 커트라인(1.010배) 돌파 시 잔량을 덤핑하며, 주가가 1층 타점 이상을 유지 중이면 1층 잔여 물량까지 시장 매수호가로 100% 폭격하여 지층을 완벽히 소각합니다.
💎 애프터마켓 3% 로터리 덫 (Lottery Trap) : 정규장 마감 직후(16:05 EST / 05:05 KST), 유동성 공백과 실적 발표 노이즈를 활용하여 잔여 물량 100%를 총 평단가 대비 +3% 타점에 장후 지정가(AFTER_LIMIT)로 자동 셋팅하여 극단적 꼬리 수익을 포획합니다.
💎 삼위일체 소각 엔진 및 마이너스 손절 개방 : 텔레그램 수동 리셋 격발 시 본장부, 가상장부(Escrow), LIFO 큐(Queue), 숨겨진 백업본까지 4중 찌꺼기 데이터를 100% 동시 소각합니다. KIS 실잔고 0주 스캔 시 누적 수익이 마이너스라도 즉각 장부를 비우고 새출발 모드로 락다운을 강제 해제합니다.
💎 스마트 장부 동기화 엔진 (TrueSync) : 미국장 영업일 스케줄을 자동으로 인식하며, 과거 데이터를 덮어쓰지 않고 오차만큼만 교정하는 비파괴 보정(CALIB) 및 야후 연동 액면분할 자동 소급 로직을 지원합니다.
🛠️ 설치 및 실행 방법 (Installation & Usage)
📌 1. 필수 환경 (Requirements)
✔️ Python 3.12 이상
✔️ 한국투자증권 Open API 발급 (App Key, App Secret)
✔️ Telegram Bot Token 및 Chat ID
📌 2. 패키지 설치
👉 pip install requests yfinance pytz pandas_market_calendars python-dotenv pillow "python-telegram-bot[job-queue]"
📌 3. 환경 변수 설정 (.env 파일 생성)
프로젝트 최상단 폴더에 .env 파일을 만들고 아래 양식에 맞게 본인의 키를 입력합니다.
👉 TELEGRAM_TOKEN=나의_텔레그램_봇_토큰
👉 ADMIN_CHAT_ID=나의_텔레그램_채팅방_ID숫자
👉 APP_KEY=나의_한국투자증권_APP_KEY
👉 APP_SECRET=나의_한국투자증권_APP_SECRET
👉 CANO=나의_계좌번호_앞8자리
👉 ACNT_PRDT_CD=01 또는 22
📌 4. 프로그램 실행
👉 python main.py
(권장: 서버 환경에서는 nohup python main.py & 명령어를 사용하여 백그라운드에서 24시간 가동되도록 설정하세요.)
📂 파일 구조 (Directory Structure)
📁 main.py : 스케줄러 구동, 의존성 주입(DI) 라우팅 및 프로그램의 메인 진입점.
📁 scheduler_core.py : API 토큰 갱신, 장부 동기화, 시스템 자정 작업 등 생명 유지를 전담하는 백그라운드 관리 코어.
📁 scheduler_trade.py : 정규장 LOC 장전, 스나이퍼 감시, V-REV 전용 1분봉 슬라이싱 및 스윕 피니셔, 애프터마켓 로터리를 전담하는 전투 타격 스케줄러. (자동 긴급 수혈 로직 영구 적출 완료)
📁 strategy.py : 2대 코어로 연산을 분배하고 폐기된 레거시 모드를 강제 교정(Auto-Upgrade)하는 중앙 라우터 엔진.
📁 strategy_v14.py : 오리지널 무한매수법(V14) 연산 로직이 100% 캡슐화된 독립 플러그인.
📁 strategy_reversion.py : V-REV 역추세 하이브리드 타점 산출, 공수 스위칭 및 고유 단가 기반 독립 탈출 연산 엔진.
📁 queue_ledger.py : V-REV 날짜 기반 지층 통합 LIFO 큐 데이터 비파괴 관리 모듈.
📁 volatility_engine.py : 1년 ATR 평균 및 VXN/HV 기반 시장 변동성 마스터 스위치 연산 모듈.
📁 broker.py : 한국투자증권 API 통신 및 야후 파이낸스 데이터(데드락 방어 적용) 가공 클래스.
📁 telegram_bot.py / telegram_view.py : 레거시 UI를 완벽히 적출한 초경량 텔레그램 봇 컨트롤러 및 통합 지시서 렌더링 로직. 수동 긴급 수혈(Emergency MOC) 3중 경고 방어막 및 콜백 라우터 탑재.
📁 config.py : 각종 JSON 데이터를 저장하고 불러오는 로컬 캐싱 및 설정 매니저.
📁 version_history.py : 코드 업데이트 최신 히스토리 기록.
