# 🚀 KIS-API-Python-Trading-Bot-Example (V26.02 4대 집행 모드 통합 에디션)  
본 프로젝트는 한국투자증권(KIS) Open API를 활용하여 미국 주식 자동매매 시스템을 구축해보는 파이썬(Python) 예제 코드입니다.  
이 코드는 증권사 API 통신 방법과 스케줄러 자동화, 텔레그램 봇 제어 등을 학습하기 위한 기술적 레퍼런스로 작성되었습니다.

## ✨ 주요 기술적 특징 (Key Features)  
💎 **집행 모드 4원화 아키텍처 (V26.02)**: V14(오리지널 무매)와 V-REV(역추세) 각각에 대해 초안정성 'LOC 단일 타격' 방식과 시장 미시구조 추종형 'VWAP 타임 슬라이싱' 방식을 도입하여 총 4개의 독립된 퀀트 집행 모드를 지원합니다.  
💎 **수동 VWAP 시그널 모드 탑재 (수수료 방어)**: 증권사 API 수수료를 100% 방어하기 위해 봇을 시그널 참모로 전환하는 수동 모드를 신설했습니다. 자동 주문(EXEC)을 락다운하고 텔레그램 지시서를 통해 장 마감 30분 전 수동 세팅 가이던스 및 치명적 경고망을 렌더링합니다.  
💎 **V14 VWAP 유동성 추적 엔진 이식**: 오리지널 무한매수법의 강력한 T값 및 예산 공식을 유지하면서, 장 마감 30분 전(15:30 EST)부터 1분 단위 유동성 곡선(U-Curve)을 따라 예산을 분할 사냥하는 정밀 슬라이싱 플러그인을 결합했습니다.  
💎 **타임라인 시프트 및 유령 매수 차단(V25.26)**: VWAP 슬라이싱 타임 윈도우를 '장 마감 33분 전 ~ 4분 전(04:27~04:57 KST)'으로 앞당겨 물리적 미체결 맹점을 원천 차단했습니다. 또한 0주 졸업 달성 시 즉각 스케줄러를 동결시키는 'Daily Buy-Lock'을 이식하여 유령 매수를 100% 억제합니다.  
💎 **스윕 피니셔 거래소 락다운 우회 디커플링**: 장 마감 2분 전(15:58 EST) 스윕 덤핑 시 미국 거래소 취소 불가 규정으로 묶인 수동 긴급 수혈(MOC) 물량을 피하기 위해 KIS 실시간 원장을 재스캔합니다. 전체 수량이 아닌 '순수 매도 가능 수량(ord_psbl_qty)'만을 정밀 타격하며, 락다운 시 덤핑을 자동 생략(Skip)하는 무결점 방어막이 가동됩니다.  
💎 **핫픽스 그랜드 슬램 및 런타임 붕괴 방어**: 빈 장부 스캔 시 발생하는 런타임 붕괴 및 Windows 권한 락(Lock)을 차단하고, 토큰 만료 시간, 자정 래핑, 서머타임 프리마켓 타임존 오판 등 시간 연산 맹점을 수학적으로 완벽히 교정했습니다.  
💎 **차세대 AVWAP 하이브리드 암살자 엔진**: V-REV 본대와 100% 독립된 물량/예산 격리 방화벽을 구축했습니다. 잉여 현금 100%를 무기화하여, 듀얼 레퍼런싱(Dual-Referencing) 기반의 기초자산 팩트를 스캔하고 파생상품 호가창에 즉각 타격하는 투트랙 암살 전술이 탑재되었습니다.  
💎 **리버스 모드 절대 하드스탑 확정 탈출**: 장중 휩소(Whipsaw)에 의한 무한 스위칭 API 과부하를 원천 방어합니다. 매일 시스템 초기화 시점(17:00 KST)에 단 한 번만 TQQQ(-15%) / SOXL(-20%) 회복 여부를 팩트 스캔하여 가변 변수 의존 없이 V14 본대로 확정 복귀합니다.  
💎 **V14 대박익절 무결점 스윕 피니셔**: 대박 익절(리버스 생략) 조건 달성 시, 100% 지정가 전량 덤핑 후 즉시 루프를 탈출(Early Return)하여 단 1주의 물량도 야간 시장 하락 리스크에 노출되지 않도록 누수 버그를 영구 차단했습니다.  
💎 **V-REV 층별 고유 단가 독립 탈출 (Decoupling)**: 기존의 전체 평단가 의존 로직을 폐기하고, 각 매수 지층(Layer)이 본연의 고유 진입 단가를 기억하여 조건 충족 시 독립적으로 개별 탈출하는 후입선출(LIFO) 아키텍처를 완벽 이식했습니다.  
💎 **수동 긴급 수혈 (Emergency MOC) 시스템**: 통신 지연에 따른 자동 스케줄러의 오판 및 강제 청산 엣지 케이스를 차단했습니다. 텔레그램 큐(Queue) 관리 메뉴에서 3중 경고를 거쳐 최근 물량(LIFO)을 시장가(MOC)로 강제 매도하여 안전하게 실탄을 확보합니다.  
💎 **스마트 장부 동기화 엔진 (TrueSync)**: 미국장 영업일 스케줄을 자동으로 인식하며, 과거 데이터를 덮어쓰지 않고 오차만큼만 교정하는 비파괴 보정(CALIB) 및 야후 연동 액면분할 자동 소급 로직을 지원합니다.  
## 🛠️ 설치 및 실행 방법 (Installation & Usage)  
📌 **1. 필수 환경 (Requirements)**  
✔️ Python 3.12 이상  
✔️ 한국투자증권 Open API 발급 (App Key, App Secret)  
✔️ Telegram Bot Token 및 Chat ID  
📌 **2. 패키지 설치**  
👉 pip install requests yfinance pytz pandas_market_calendars python-dotenv pillow "python-telegram-bot[job-queue]"  
📌 **3. 환경 변수 설정 (.env 파일 생성)**  
프로젝트 최상단 폴더에 .env 파일을 만들고 아래 양식에 맞게 본인의 키를 입력합니다.  
👉 TELEGRAM_TOKEN=나의_텔레그램_봇_토큰  
👉 ADMIN_CHAT_ID=나의_텔레그램_채팅방_ID숫자  
👉 APP_KEY=나의_한국투자증권_APP_KEY  
👉 APP_SECRET=나의_한국투자증권_APP_SECRET  
👉 CANO=나의_계좌번호_앞8자리  
👉 ACNT_PRDT_CD=01 또는 22  
📌 **4. 프로그램 실행**  
👉 python main.py  
(권장: 서버 환경에서는 nohup python main.py & 명령어를 사용하여 백그라운드에서 24시간 가동되도록 설정하세요.)  
## 📂 파일 구조 (Directory Structure)  
📁 main.py : 시스템 진입점, 스케줄러 등록 및 봇 데몬 구동 코어  
📁 scheduler_core.py : API 토큰 갱신, 장부 동기화, 시스템 자정 작업 및 리버스 모드 일일 1회 확정 탈출 팩트 스캔을 전담하는 백그라운드 관리 코어  
📁 scheduler_trade.py : 정규장 LOC/VWAP 장전, 스나이퍼 감시, 1분봉 슬라이싱 및 스윕 피니셔, 애프터마켓 로터리를 전담하는 전투 타격 스케줄러  
📁 strategy.py : 4대 집행 모드(V14 LOC/VWAP, V-REV Auto/Manual)로 연산을 분배하고 레거시 모드를 강제 교정하는 중앙 지능형 동적 라우터(Dynamic Router)  
📁 strategy_v14.py : 오리지널 무한매수법(V14) 퀀트 공식 및 초안정성 LOC 단일 타격 전술이 캡슐화된 독립 플러그인  
📁 strategy_v14_vwap.py : **(NEW)** V14 공식 기반 장 마감 30분 전 1분 단위 유동성 추적(VWAP) 분할 타격 전술 플러그인  
📁 strategy_reversion.py : V-REV 역추세 하이브리드 타점 산출, 공수 스위칭 및 고유 단가 기반 독립 탈출 연산 엔진  
📁 strategy_v_avwap.py : 잉여 현금을 100% 활용하여 당일 VWAP 기반 딥매수 및 스퀴즈 사냥을 전담하는 투트랙 암살자 엔진  
📁 queue_ledger.py : V-REV 날짜 기반 지층 통합 LIFO 큐 데이터 비파괴 관리 모듈  
📁 volatility_engine.py : 1년 ATR 평균 및 공포지수(VXN/HV) 동적 연산, 가중치 산출 레이더  
📁 broker.py : KIS API 통신 및 야후 파이낸스 데이터 가공, 범용 1분봉 스캔 엔진(get_1min_candles_df) 탑재 클래스  
📁 telegram_bot.py / telegram_view.py : 초경량 텔레그램 봇 컨트롤러 및 2단계 뎁스 모드 선택(3-Way 컨트롤 타워) 렌더링 로직  
📁 config.py : JSON 데이터베이스 입출력, 에스크로 동적 캐싱 및 수동/자동 모드 플래그 관리자  
📁 version_history.py : 코드 업데이트 최신 히스토리 기록
